# tRPC Utils Invalidation - User Stories

## Overview

This technical story focuses on refactoring client-side cache invalidation to use tRPC `useUtils` helpers instead of raw TanStack query keys. The goal is to keep invalidation type-safe and resilient to procedure renames across the client.
