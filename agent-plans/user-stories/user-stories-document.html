Domain 08: P2P Reservation Confirmation

Overview

Enhances the simplified paid reservation flow with TTL countdown, explicit payment instructions, T&C acknowledgement, optional payment proof submission, and owner proof review. Includes expiration handling driven by cron jobs and UI states for expired reservations.
