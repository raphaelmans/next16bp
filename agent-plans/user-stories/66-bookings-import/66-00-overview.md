# Bookings Import - User Stories

## Overview

The Bookings Import domain helps organization owners migrate already-booked reservations from external systems (calendar exports or spreadsheets) into KudosCourts so that availability reflects real-world bookings and prevents double-booking.

This domain focuses on **importing booked time ranges** and presenting them in a review UI that matches KudosCourts' internal data structure.
