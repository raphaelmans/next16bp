# Place Verification - User Stories

## Overview

The Place Verification domain defines how an organization owner can request verification for a specific **place** (venue listing) before enabling reservation support.

Unverified places remain publicly discoverable (so the venue can be found), but are not bookable until:

1. The place is verified by an admin, and
2. The owner explicitly enables reservation support.
