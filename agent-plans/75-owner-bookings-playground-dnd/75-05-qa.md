# QA Checklist

## Happy Paths

- Drag 1h maintenance preset onto 08:00 -> block is created.
- Drag 2h maintenance preset onto 18:00 -> block is created 18:00-20:00.
- Drag 1h walk-in preset onto 09:00 -> walk-in is created.

## Undo

- After create, click Undo -> block is cancelled and disappears.
- Undo after navigation/invalidation still cancels the correct block.

## Conflicts

- Drop a block on an overlapping time -> creation fails with clear error toast.
- Drop a block over an existing reservation -> creation fails (reservation overlap error).

## Time Zone

- Timeline labels match `place.timeZone`.
- Created blocks appear at the correct local time after refresh.

## Accessibility + Input

- Keyboard drag works for presets (or has a non-drag fallback: click preset -> click time).
- Touch drag works with press delay.
- `prefers-reduced-motion` does not cause distracting transitions.

## Import Overlay (Phase 4)

- Open studio with `jobId` -> banner + draft list appears.
- Drag a draft row onto a new time -> row updates and status recalculates.
- Commit disabled when errors exist; enabled when errors are resolved.
