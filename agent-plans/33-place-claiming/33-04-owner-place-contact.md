# Phase 4: Owner Edits Place Contact Details

**Dependencies:** Phase 1 complete  
**Parallelizable:** Yes  
**User Stories:** US-17-05

---

## Objective

Allow owners to edit place contact details after claim, and ensure those changes reflect on the public place page.

---

## Modules

### Module 4A: Owner Place Form Supports Contact Details

#### Owner UI

- Add a "Contact" section to the owner place form with:
  - Website URL
  - Facebook URL
  - Instagram URL
  - Viber info
  - Other contact info

#### Behavior

- Pre-fill existing contact info when editing
- Save updates
- Public page reflects updated contact info

---

## Testing Checklist

- [ ] Owner can update contact details.
- [ ] Unauthorized user cannot update contact details.
- [ ] Public place page shows updated contact info.
