## Post-Deployment Verification

### Meta Tag Verification

```bash
# Check venue page meta tags
curl -sL https://www.kudoscourts.com/venues/kitchenline-pickleball-center | grep -E "<title>|canonical|og:" | head -10

# Verify no "undefined" anywhere
curl -sL https://www.kudoscourts.com/venues/kitchenline-pickleball-center | grep -i "undefined"
# Should return nothing
```

### Sitemap Verification

```bash
# Check sitemap has www URLs
curl -sL https://www.kudoscourts.com/sitemap.xml | grep "https://" | head -5

# Check venue pages in sitemap
curl -sL https://www.kudoscourts.com/sitemap.xml | grep "/venues/" | head -5

# Validate XML
curl -sL https://www.kudoscourts.com/sitemap.xml | xmllint --noout -
```

### Structured Data Verification

```bash
# Check for ld+json
curl -sL https://www.kudoscourts.com/venues/kitchenline-pickleball-center | grep "application/ld+json"
```
