# Client + Server Migration Plans (Agent-Plan Artifacts)

## Goal
Create coordinated agent-plan artifacts for:
1) Server migration to move all server-only code under `src/lib/`.
2) Client migration to align with guides (common utilities, feature `hooks.ts`/`schemas.ts`, and composability refactors).

## Scope
- Focus: planning artifacts only (no code changes).
- Output: two new agent-plan domain folders with overview, phases, and developer checklists.
- Include composability refactors for top offenders (bookings, place detail, owner availability, admin court pages, bookings import review).
- Out of scope: docs under `docs/apply-architecture/` (user chose agent-plan artifacts only).
