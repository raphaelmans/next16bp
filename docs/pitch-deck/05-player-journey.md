# Player Journey

Discovery to booking

1. Discover places by sport + location (and optionally amenities).
2. Open a place detail page and choose:
   - A specific court, or
   - "Any available" (best price option).
3. Choose duration (60/120/180+), then select a start time.
4. Request booking (clear reservation status + next steps).

Paid bookings (P2P-style)

- The platform does not process payments.
- Players receive venue payment instructions in the reservation context.
- Payment is timeboxed, then the player marks payment and the owner confirms.
