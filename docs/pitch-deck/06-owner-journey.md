# Owner Journey

From setup to daily ops

1. Create an organization.
2. Add a place (venue) and one or more courts.
3. Configure:
   - Day-specific hours (including overnight)
   - Hourly pricing rules
4. Publish 60-minute slots (inventory).
5. Manage reservations:
   - Inbox-first workflow
   - Court/place filters
   - Fast confirm/reject actions

Payment methods (org-scoped)

- Owners can define one or more payment methods (wallet/bank), with per-method instructions.
- A single default method is highlighted to reduce player confusion.
