# 8-Second Social Video Script (Vertical)

Audience: venue owners/operators (sports courts)

Goal: get owners to claim + verify their venue so reservations can be enabled.

## Primary Version (8s)

Format: 9:16, jump cuts, big captions, show a simple calendar UI + "Verified" badge.

Design system styling (KudosCourts):

- Background: warm neutral `#FAFAF9`; cards `#FFFFFF`; borders `#E7E5E4`; soft shadow (subtle, warm)
- Primary (CTA): Teal `#0D9488` (hover/darker `#0F766E`); primary tint `#CCFBF1`
- Accent (highlights/links): Orange `#F97316` (optional gradient `#FB923C` -> `#F97316`)
- Success (available/confirmed): `#059669` with tint `#ECFDF5`
- Typography for captions:
  - Headline words: Outfit 700-800 (tight tracking, ~`-0.02em`)
  - Supporting labels: Source Sans 3 600 (clean + readable)
- Shape: rounded bento cards (radius ~16px), pill badges (radius 9999px)
- Motion: quick staggered fade-in-up for UI cards; fast 150-200ms cuts between beats

0.0-1.2s

- Visual: fast montage of DMs / missed calls / messy notes.
- On-screen text: "Bookings stuck in DMs?"

- Styling cue: available slots use success tint (`#ECFDF5` / `#059669`); selected slot uses primary tint (`#CCFBF1` / `#0F766E`).

- VO: "KudosCourts turns courts into bookable slots."

- On-screen text: "Claim + verify (phone + socials)"

5.8-8.0s

- Visual: "Reservations: ON" + "6 months free" stamp.
- Styling cue: "Reservations: ON" in teal; "6 months free" callout in orange (accent), not a second teal button.

- VO: "First 100 verified: six months free. DM 'VERIFY'."
