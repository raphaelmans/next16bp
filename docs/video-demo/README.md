# Video Demo Scripts

This folder stores marketing and product demo scripts for KudosCourts.

Guidelines:

- Use the brand position: `Court discovery for players. Free reservation system for venues.`
- Avoid any metric claims we cannot verify.
- Keep `Free for venues` copy tied to platform usage only (never court-time pricing).
- Maintain the current visual language: minimalist bento, warm neutrals, teal CTA, orange accents.
