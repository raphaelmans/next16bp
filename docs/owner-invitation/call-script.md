# Call / WhatsApp Script (Owners)

## 60–90s script

“Hi, is this {{first_name}}? I’m {{your_name}} from KudosCourts.

We help players discover courts, and we provide a free reservation system for venues—so bookings aren’t stuck in DMs and spreadsheets.

{{#if listing_url}}
We created a read-only listing for {{venue_name}} so players can find it. If you manage the venue, the next step is to claim + verify it so you can enable online reservations.
{{/if}}

Verification is quick: phone + official social links (and documents only if needed). You control when reservations go live.

We don’t process payments—players pay venues directly using your existing payment methods.

Would you like the link to claim + verify, or would you prefer we remove the listing?”

## Common objections (fast replies)

- “We already take bookings via Facebook/phone.”
  - “Keep that. This just standardizes requests, shows availability, and reduces double bookings.”
- “Is it really free?”
  - “Yes—core listing and reservations are free for venues. We aim to sustain via ads/sponsored placements, not subscriptions.”
- “Why verification?”
  - “To prevent fake owners from claiming your venue and to build trust for players.”
- “We don’t want to be listed.”
  - “No problem—reply REMOVE and we’ll take it down immediately.”

## Close options

- “Can I send the link now via Messenger/WhatsApp?”
- “What’s the best email/number to send the steps?”
