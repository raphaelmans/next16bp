# Owner Invitation (Sales/Outreach Templates)

Use these templates to invite venue owners/operators to list/claim and verify their venue on KudosCourts.

## Primary CTA link

https://www.kudoscourts.com/owners/get-started

## Personalization checklist

- Venue name: {{venue_name}}
- City/area: {{city}}
- Sport: {{sport}}
- Listing URL (if already curated): {{listing_url}}
- Best contact channel: {{preferred_channel}}

## Messaging rules

- Say “free for venues” (platform), not free court time.
- Say “rates” / “price” (venue charges), not SaaS “pricing”.
- No payment processing: players pay venues directly.
- No fake stats or fabricated testimonials.
- Always include an opt-out path if we curated a listing.

## Two outreach tracks

1. Track A — We already curated a read-only listing
   - Send: “We added a read-only listing for {{venue_name}}. Want to claim + verify it?”
   - Include listing link + opt-out option.
2. Track B — Not curated yet
   - Send: “Want to list your venue for free and accept online reservations?”
   - Include setup link.

## Small-commitment CTA (recommended)

- “Reply VERIFY and I’ll send the setup link.”
- “Comment VERIFY and I’ll DM you the link.”

(Commitment & consistency; reduces activation energy.)
