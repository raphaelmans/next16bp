# Brand Positioning — KudosCourts as a Free Reservation System

## Positioning Statement

KudosCourts is a **free reservation system** and curated discovery platform for sports venues in the Philippines—built to help small/new venues stop managing bookings in spreadsheets, DMs, and calendars, and help players find real availability and reserve time faster.

## Target Audiences

### Players

- Want: real availability, less back-and-forth, faster booking confirmation
- Current workaround: FB groups/DMs, screenshots, outdated pins

### Venue owners / operators (especially small/new venues)

- Want: fewer double bookings, less admin, a simple setup with no contracts
- Current workaround: Excel sheets, notebooks, Google Calendar, DMs

## Core Promise (Honest)

- **Free to use for venues** (core listing + reservation workflows)
- **No payment processing**: venues keep existing payment methods; players pay venues directly
- **Sustainable**: we plan to monetize via advertising/sponsored placements (transparent)

## Differentiators

- “Listed → claim → verify → enable reservations” trust pipeline
- Import existing bookings to prevent conflicts
- Built-in chat so players and venues can coordinate

## Guardrails (What we must not imply)

- “Free” does NOT mean court time is free; venues can charge rates.
- Avoid “Pricing” (SaaS plan) language on public pages; use **rates** or **price**.
- Don’t invent stats/testimonials. Use in-product counters only when they’re real.
