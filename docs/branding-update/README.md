# Branding Update — “Free Reservation System” (KudosCourts)

## Goal

Make it unmistakable that KudosCourts is:

- A court discovery + booking experience for players, and
- A **free reservation system** for venue owners (Philippines)

## Why

“Pricing” language (even when meant as venue rates) can imply SaaS subscriptions.
This update removes that confusion and aligns site copy with the new promise: **free core, sustained via ads**.

## Decisions (Locked)

- Positioning: Balanced (players + venues)
- Free promise: Free core, sustainable via ads (transparent)
- Footer “Pricing” link: removed
