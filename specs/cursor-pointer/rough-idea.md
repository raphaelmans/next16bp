# Cursor Pointer — Rough Idea

Ensure all buttons and clickable shadcn-ui components use `cursor-pointer`. This isn't limited to `<Button>` — it covers any shadcn component that a user clicks: toggles, switches, selects, checkboxes, radio buttons, tabs, dropdown menus, context menus, calendar day cells, etc.

Use the shadcn MCP to analyze upstream component source and identify which components need the fix.
